lab 3 - WAF
##########################################################

NGINX App Protect provides web application firewall (WAF) security protection for your web applications,
including OWASP Top 10, bot protection, JSON & XML schema validation...

This lab explains how to deploy NGINX App Protect as well as configure App Protect and upgrade the App Protect signature sets.
